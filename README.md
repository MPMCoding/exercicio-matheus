# Exercício TMDB II - Catálogo de Filmes

Este é um projeto de catálogo de filmes dinâmico que consome dados em tempo real da API do The Movie Database (TMDB). O design foi inspirado com foco em uma experiência de usuário fluida e moderna.

## 🚀 Funcionalidades

- **Integração com TMDB**: Listagem de filmes em cartaz e próximos lançamentos no Brasil.
- **Busca em Tempo Real**: Filtro inteligente de filmes por título ou descrição.
- **Design Premium**: Interface escura com carrosséis horizontais (Scroll Snap).
- **Detalhes Dinâmicos**: Seção de detalhes estilo "cinema" que aparece ao clicar em um poster.
- **Responsividade**: Totalmente adaptado para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- [TMDB API](https://www.themoviedb.org/documentation/api)

## 📦 Como usar

1. Clone o repositório.
2. Abra o arquivo `trabalho-tmdb.html` no seu navegador.
3. Certifique-se de que o `script.js` possui um Token de Acesso válido da API TMDB.
